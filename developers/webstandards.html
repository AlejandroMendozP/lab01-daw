El Estándar: HTML5 Semántico

La W3C (World Wide Web Consortium) establece que el código debe ser semántico para mejorar la accesibilidad y el SEO.

¿Por qué es importante?

Permite que los motores de búsqueda y las tecnologías de asistencia (como lectores de pantalla) entiendan la estructura del contenido.

  • <header>: Define la cabecera.
  • <main>: Contenido principal único.
  • <footer>: Pie de página con autoría.
